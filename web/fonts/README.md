# Fuentes - DynaPuff

## Fuente actual: DynaPuff (Google Fonts)

La fuente **DynaPuff** se carga automáticamente desde Google Fonts. No se requieren archivos locales.

### Configuración

- **Fuente**: DynaPuff
- **Peso**: 600 (Semi-bold)
- **Fuente**: Google Fonts CDN
- **Aplicada a**: Todos los títulos "Pinot" con la clase `.font-branding`

### Cambiar la fuente

Para cambiar a otra fuente, edita `web/css/fonts.css` y actualiza:
- El `@import` de Google Fonts
- Las clases `.font-branding`, `.font-title`, `.font-logo`
